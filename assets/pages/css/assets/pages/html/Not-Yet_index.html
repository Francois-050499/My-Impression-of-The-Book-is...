[JSなし!]CSSで実現、ボタンホバーエフェクトまとめ

なくてもいいかもしれないが、あれば無味乾燥なページを少しでもリッチなページにしてくれるホバーエフェクトについてまとめました。

ホバー (:hover)とは、「対象物に対してマウスカーソルを重ねることで、自動で何らかの処理が行われること」という意味です。

ボタンホバーエフェクトまとめ、Part.2

Part.2では、疑似要素を用いたエフェクトや、全体的にボタンにデザインがついたものについてのエフェクトをまとめました。

See the Pen Button Hover Effect Part.2 by Francois (@Franc01S-050400) on CodePen.

[ボタン1] ボタンの輪郭が強調される その1

ポイント

hoverによる疑似要素のopacityの変更と、ボタンの輪郭のみを別で疑似要素を用意することによって、輪郭が強調される

[ボタン2] ボタンの輪郭が強調される その2

ポイント

hoverによるopacityの変更をその1の逆にすることで実現できると思ったが、出来なかったため、hover時にwidthを100%にすることで解決した。

[ボタン3] ボタンの下に影ができる

ポイント

疑似要素で、radial-gradient、ellipse at center による中央から広がっていく影を表現し、hoverの時に出現するようにopacityで設定し、影が留まるようにtransformYをつけている。

[ボタン4] ボタンが鼓動する

ポイント

transform scaleを1より大きくしたり、小さくしたりすることをanimationで設定することによって実現させている。

[ボタン5] ボタンからきらめきが出ている

ポイント

outline-color、outline-offsetを使うことで、ここでは12pxまでoutlineが広がった後消えるようにすることできらめきを表現している。

[ボタン6] ボタンの文字が回転する

ポイント

spanタグで回転させたい文字を囲って、transformX方向で360°回転させている。Y軸方向にも回転させたり、spanタグで囲い方を工夫することでたくさんの表現ができる (後から出てくる)

[ボタン7] ボタンの枠線を変更する

ポイント

borderの線をsolidから、dashedに変更することで、変更している。dottedだったり、二重線だったりで応用できるので枠線の変更はこれだけにした。

[ボタン8] グラデーションがかかる変化

ポイント

background-imageでグラデーションを、background-sizeで大きくすること、background-positionで、おおきくしたところを覗き込むようなかたちにして表現している。

[ボタン9-15] ボタンの背景をつけて、変化をつける

ポイント

(repeating-)linear-gradientや、これらを複数用いることによって縞模様や市松模様、水玉模様を作ることができる。また、linearや、cubic-bezierを用いて、速さを変えてみたり、模様をanimationで変化をつけたりして自分なりに仕上げた。これらはグラディエーターがあるため作ることができるが、アーガイル柄などの発展形も作ることができるため時間のある人は試してみてほしい。

[ボタン16] ボタンの背景をつけて、変化をつける (疑似要素を使った方法)

ポイント

90°方向の縞模様をanimationで動かそうとしたときに、なぜか動かなかったために疑似要素を用いて、縞模様を動かす方法を実装した。方法は今までと変わらない疑似要素の使い方で出来る。

[ボタン17] 文字の感覚を広げる

ポイント

letter-spacingを用いることによって文字の間隔を広げることができる。また、間隔を詰めることもできるため、ボタンだけではなく、強調したい文字に対して使う応用もできると思う。

[ボタン18] 文字の一部を大きくする

ポイント

大きくしたい文字に対してspanタグをつけて、hover時に大きくする方法もあるが、::first-letterという、最初の文字に効果を与えるというものを用いて表現した。hoverに文字がずれるのが気になったため、line-heightを少し変更している。

[ボタン19] 文字の一部を変更する

ポイント

spanタグで変更したい文字を囲い、Y軸方向に変化する効果をつけている。

[ボタン20]文字が流れるように変化する

ポイント

spanタグで変更したい文字を囲い (この場合はひとつずつ)、Y軸方向に変化する効果をつけている。その後ひとつずつになめらかな変化をつけるために、nth-childを用いて、効果をtransition-delayで遅らせることによって連続して変化しているようにみえる。

なぜ、この題目を選んだのか？

リッチなページをJavaScriptを用いずにどう表現するかに対して自分の引き出しを増やしたかった。仮に実務で使えないとしても、なぜ使われないか考えることも大事なのではないかと思った。
