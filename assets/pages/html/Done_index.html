ボタンホバーエフェクトまとめ、Part.1

Part.1では、比較的ひとつのプロパティで実装できるエフェクトと、疑似要素を用いたエフェクトを紹介します。

See the Pen Button Hover Effect Part.1 by Francois (@Franc01S-050400) on CodePen.

[ボタン1] ホバーすると薄くなる

ポイント

opacityの値を少し小さくすることによって、ホバーしたときに分かるようにしている。

[ボタン2] ホバーすると色が変化する

ポイント

colorの値をデフォルトのカラー(ここでは赤)を変更している。

[ボタン3] ボタンの色が反転する

ポイント

background-colorとcolorプロパティを反転するように設定している。難しいことはしていない。

[ボタン4] ボタンが拡大する

ポイント

transform scaleを1より大きくすると拡大し、1より小さくすると縮小する。

[ボタン5] 角丸になる

ポイント

border-radiusをデフォルトのボタンからなくすことで可能になる、逆も応用可能。

[ボタン6] ボタンを押しているような変化

ポイント

box-shadowをデフォルトのボタンに追加し、ホバー後shadowをなくし、Y軸方向に対して変化をつけることで実現させている。

[ボタン7] グラデーションの変化

ポイント

background-sizeと、positionを使うことで大きくしたボタンに対してデフォルトのボタンを上から見たような変化がグラデーションの変化に見えている。池の中にゴーグルで覗いている感覚。

[ボタン8] ボタンが光ったような変化

ポイント

animationとcolorの変化を使うことで実現できる。animationの%(パーセント)を不規則にして暗めの色を挟むことでできる比較的簡単な変化

[ボタン9-13] ある起点から塗られるような変化

ポイント

疑似要素の登場と、transform-originの考え方が大切である。X軸とY軸の考え方が独特なので気を付ける。

[ボタン14-16] ある起点から塗られるような変化 +α

ポイント

疑似要素の登場と、overflow:hiddenを用いることではみ出した部分は見えないようにしている。

[ボタン17] ボタンを傾かせる変化

ポイント

transform rotateでの数字を変化することで実践している。また、animationで上記のボタン4などと併用することで応用が予想できる。

[ボタン18] 角丸を丸くする変化

ポイント

角丸を変更するホバーエフェクトは多く、ボタン5の反対であるが、採用した。

[ボタン19-20] 外側(内側)からボタンの枠がでてくる変化

ポイント

疑似要素でホバーしたときに変化が起こるようにして、scaleは1より大きくすることで外側に、小さくすることで内側から枠線が出現しているように見える。transitionにopacityの追加によりフワッと動くように工夫している。

なぜ、この題目を選んだのか？

リッチなページをJavaScriptを用いずにどう表現するかに対して自分の引き出しを増やしたかった。仮に実務で使えないとしても、なぜ使われないか考えることも大事なのではないかと思った。
